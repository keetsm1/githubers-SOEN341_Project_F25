Clone & run
1) Clone/pull the repo
2) In your code editor, install dependencies:
   npm ci
3) From Sprint2/squad-events-pro/ run:
   npm run dev

The app will start on:  http://localhost:8080/

Notes:
If Supabase env vars are not set, the app will let you login (use mock data provided).
Most of the features are hardcoded.
Read DB_SCHEMA_PROPOSAL.txt for database details.(for Gunkeet and Arnav)
Change example.gitignore.txt to .gitignore
